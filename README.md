# SAMP

Este é um sistema para gerenciamento de marcas e patentes que podem ser extraídos do RPI/INPI, orgão resposável pela emissão desses dados aqui no brasil.

A idéia é criar uma conexão maior dentre o entermediador dessas operações e as pessoas detentores desses processos. 

Será um sistema fechado para um grupo de pessoas que possui a necessidade de automatizar esse processo.
